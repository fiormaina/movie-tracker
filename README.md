# Movie Tracker

Монорепозиторий проекта Movie Tracker.

## Структура проекта
- `movie-tracker-front/` - клиентская часть приложения
- `movie-tracker-back/` - серверная часть приложения
- `movie-tracker-extension/` - браузерное расширение для отслеживания просмотра

## Рекомендованный порядок запуска
1. Поднять backend.
2. Запустить frontend.
3. Собрать и загрузить extension в браузер.
